**Prerequisite:**

Read using shell scripting

### Bash Read Styntax:

```jsx
read <options> <arguments>
```

**example1:**

```jsx
#!/bin/bash
read -p "What is your name: " name
read -p "Enter the password: " -s password
echo $name
echo $password
read -p "It will timeout in 10 seconds: " -t 10
```

**While loop Syntax to read a file:**

```jsx
#!/bin/bash
while read -r line;

   echo "$line";

```

```jsx
#!/bin/bash
echo "Welcome to service status check script "
read -p "Enter the service name to check its status: " service_name
if [ -z $service_name ];
then
     echo " Please enter a valid service name "
else
     systemctl status $service_name
fi
```
